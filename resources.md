# Resources

## Tools Used in This Course

- GitHub Pages (Markdown)
- Fusion 360 CAD software
- Inkscape 2D Design Software
- Prusa Slicer
- Prusa i3 MK3s 3D printers
- Soldering Station
- Arduino IDE
- Sewing Machines

## Recommended Learning

- Observe everyday products: try to find what you like or dislike
- Practice sketching ideas
- Practice Fusion 360
- Read documentation of products
- Document everything  

**You do not need prior experience.**
