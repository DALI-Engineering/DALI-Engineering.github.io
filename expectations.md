# Expectations and Documentation

## Participation

- Attend weekly sessions  
- Ask questions  
- Engage in discussions  

This class is designed for beginners. Confusion is expected and encouraged.

---

## Documentation

Each student is strongly encouraged to maintain a **GitHub Pages portfolio**. However, it is not required.

Every week should include:
- A short description of what you worked on
- Photos or sketches
- Reflections on challenges and decisions

Clear documentation is a core engineering skill.
